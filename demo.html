how it works

you write <div class="nano-p-4 nano-bg-gray-100 nano-rounded-lg">
resolves to
padding: 1rem background-color: #f3f4f6 border-radius: 0.5rem
result
your styled element, no stylesheet needed

what's included

Spacing
padding, margin, gap — scales 0 through 8 with x/y/t/b shorthands
Colors
background + text — full gray ramp, white, black, primary, danger, success
Typography
size xs→3xl, weight, alignment, italic, uppercase, tracking, leading
Borders
border width, color variants, t/b edges, radius none through full
Layout
flex, grid, display, align, justify, wrap, gap, w-full, overflow
Extend
add any key to the styles object — it resolves instantly
